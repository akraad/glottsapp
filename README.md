# Glotts Landing Page

This folder is ready for GitHub Pages.

## Files
- `index.html`: main landing page
- `CNAME` (optional): your custom domain (one line)

## Deploy on GitHub Pages
1. Push `glotts-app/index.html` to your repository.
2. In GitHub: **Settings > Pages**.
3. Source: select your branch (for example `main`) and folder `/(root)` or `/docs` based on where you place `index.html`.
4. If you want a custom domain, add a `CNAME` file with your domain only:
   - Example content: `app.yourdomain.com`
5. Add DNS records at your domain provider based on GitHub Pages docs.

## Replace before going live
- Update download links in footer (`Google Play` / `App Store`).
- Update contact email.
- Optionally replace brand name/text with final product copy.
